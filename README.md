# Lab Exercise 4 - Deploying a Web Application on AWS EC2

## Aim

To deploy a static web application on an AWS EC2 instance and access it using the public IP address.

## Description

This project demonstrates how to host a website using Amazon Web Services EC2 virtual machine. A web server is installed on the instance, and HTML files are uploaded to make the website accessible online.

## Technologies Used

- HTML
- CSS
- JavaScript
- AWS EC2
- Apache Web Server

## Steps Performed

1. Created an EC2 instance.
2. Configured security group rules.
3. Connected to the instance.
4. Installed Apache web server.
5. Uploaded website files.
6. Accessed the website using public IP.

## Result

The web application was successfully deployed and hosted on AWS EC2.
